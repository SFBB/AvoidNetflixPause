# Avoid Netflix Pause(Manifest V3)

This is an extention that avoids Netflix pause in Chrome by simulating mouse events.

This is fully written by Gemini.

## How to Install

1. Open Google Chrome.
2. Navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked**.
5. Select the directory containing this project (`AvoidNetflixPause`).
